Esta directiva tranforma la apariencia de un componente HTML Basico a una estructura HTML del Framework implementado (Version 1.0 BOOTSTRAP 3.X)

El label o etiqueta del input esta definido por los attributos: name, label, title o placeholder o por el valor enviado en la directiva

Tambien se puede omitir el label o etiqueta pero manteniendo la apariencia o por el valor enviado en la directiva "false"
